# 📝 Lista de Tarefas - React

Aplicação de lista de tarefas desenvolvida em React, utilizando Hooks (`useState`) para gerenciamento de estado. O projeto permite adicionar e remover tarefas de forma dinâmica, oferecendo uma interface simples, intuitiva e responsiva para organização de atividades diárias.

## 🚀 Funcionalidades

- ➕ Adicionar novas tarefas  
- ❌ Excluir tarefas da lista  
- ⚡ Atualização dinâmica sem recarregar a página  
- 📱 Interface simples e responsiva
